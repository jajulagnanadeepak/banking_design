Cloud Native Digital Banking Platform

NovaPay is a cloud-native digital banking application built with React and Node.js.
It demonstrates a scalable banking architecture including authentication, accounts, transactions, and notifications.

## Features

* User Authentication (JWT)
* Account Management
* Money Transfers
* Transaction History
* Notifications
* Dockerized Backend
* RESTful APIs

## Tech Stack

Frontend

* React
* TypeScript
* TailwindCSS

Backend

* Node.js
* Express
* Prisma ORM

Database

* PostgreSQL

DevOps

* Docker
* Docker Compose
